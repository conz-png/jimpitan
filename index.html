Panduan Sinkronisasi

  1. Di Perangkat A: Buka tab "Backup & Sync" → Export semua data
  2. Transfer file ke Perangkat B (WhatsApp, email, flashdisk)
  3. Di Perangkat B: Buka tab "Backup & Sync" → Import data
  4. Data sekarang sama di semua perangkat!
